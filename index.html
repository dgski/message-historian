I Want This App
  • Fast and Lightweight.
  • Complete Search Results; unlike iMessage.
  • Calendar Tool to access Messages from an Exact Date.
  • Simple and Cross-Platform.

iMessage Search Sucks

I recently switched from Android to iPhone. I like it so far, but the message search functions on iPhone are frustrating! When I search something I only get one result per contact. What if I need some important information from old messages or want to re-visit a pleasant memory?

On Android I used an application called Textra which allowed me to search through my whole message history easily. If I searched "dog", for example, I would get a list of all messages that contain "dog"; No exceptions. I could also use a Calendar feature to view messages from specific days. I miss it. The iPhone platform has limitations so no application could replicate it on-device.

Messages can be backed up or synced with a Desktop, so instead, I searched the internet for a cheap,sime and responsive viewer for Messages on Desktop. I could not find anything. None of them had a simple interface that does one thing well: Show Me My Message History!

As a Software Developer, I have decided to investigate making an application myself. But time is valuable and I would like to see if other people would find this useful as well. Please let me know with with the Message Historian Feedback Form .

Thanks for taking the time to read this!
